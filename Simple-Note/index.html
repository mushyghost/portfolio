Plugin - Simple Note

Status : Released
Link : Asset Store

Description

Need a 3rd-party application to take a note? Why not take a note inside Unity when it simple to have it, right?

Simple Note provide you to take a note inside Unity Editor. You don’t need much time to take a note with Simple Note.

What can you do with Simple Note :

  • Full Support Unity 5
  • Create a Bunch Note
  • Create a Note at each Game Object
  • Create a Note at Script (MonoBehaviour)
  • Create a Note at Variable
  • Configuring Color (text and background)
  • Custom PropertyAttribute

So what are you waiting for? just grab it. Its FREE.
